Conexão com Xiaomi Mi Scale 2 via Notebook

Soluções para acessar os dados da sua balança a partir do computador

Método Recomendado: Script Python com Adaptador Bluetooth

Esta é a maneira mais confiável de conectar sua balança Xiaomi Mi Scale 2 a um notebook. Você precisará:

  1. Um adaptador Bluetooth 4.0+ (se seu notebook não tiver Bluetooth integrado)
  2. Python instalado no seu computador
  3. Algumas bibliotecas Python específicas

Instalação e Configuração

# Instale as dependências necessárias
pip install bluepy numpy

Script Python para Conexão Direta

import bluepy.btle as btle
import struct

class MiScale2(btle.DefaultDelegate):
def __init__(self):
btle.DefaultDelegate.__init__(self)

def handleNotification(self, cHandle, data):
measurements = self.parse_data(data)
print(f"Peso: {measurements['weight']} kg")
print(f"IMC: {measurements['bmi']}")
print(f"Gordura Corporal: {measurements['body_fat']}%")

def parse_data(self, data):
# Implementação do parser de dados da Xiaomi
# Os dados chegam em formato binário específico
weight = (data[2] + (data[3] << 8)) / 100.0

# Cálculos adicionais baseados na documentação não oficial
return {
'weight': weight,
'bmi': weight / ((1.75) ** 2), # Ajuste com sua altura
'body_fat': 20.5 # Este valor precisa ser calculado corretamente
}

# Conectar à balança
def connect_to_scale(mac_address):
try:
scale = MiScale2()
device = btle.Peripheral(mac_address, addrType=btle.ADDR_TYPE_RANDOM)
device.setDelegate(scale)

# Habilitar notificações
service = device.getServiceByUUID("0000181b-0000-1000-8000-00805f9b34fb")
characteristic = service.getCharacteristics("00002a9c-0000-1000-8000-00805f9b34fb")[0]
device.writeCharacteristic(characteristic.valHandle+1, b"\x01\x00")

print("Conectado! Suba na balança para medir...")

# Manter conexão ativa
while True:
if device.waitForNotifications(1.0):
continue

except Exception as e:
print(f"Erro: {e}")

# Substitua pelo endereço MAC da sua balança
connect_to_scale("AA:BB:CC:DD:EE:FF")

BLE Scanner

Para usuários de macOS, esta ferramenta permite descobrir dispositivos Bluetooth LE e interagir com eles.

Baixar para macOS

Como usar estas ferramentas:

  1. Instale o aplicativo no seu notebook
  2. Ligue a balança e certifique-se de que está próxima
  3. Abra o aplicativo e escaneie dispositivos Bluetooth
  4. Localize sua Mi Scale 2 (normalmente começa com "MI")
  5. Conecte-se ao dispositivo e explore os serviços
  6. O serviço da balança geralmente tem UUID: 0000181b-0000-1000-8000-00805f9b34fb

Solução Avançada: Servidor Local com Web Bluetooth

Para desenvolvedores, é possível criar um servidor local que atua como ponte entre a balança e o navegador:

# Servidor em Python usando Flask e BluePy
from flask import Flask, jsonify
from flask_cors import CORS
import bluepy.btle as btle
import threading
import time

app = Flask(__name__)
CORS(app)

# Variáveis globais para armazenar os dados mais recentes
latest_data = {}

class MiScale2(btle.DefaultDelegate):
def __init__(self):
btle.DefaultDelegate.__init__(self)

def handleNotification(self, cHandle, data):
global latest_data
latest_data = self.parse_data(data)
print(f"Dados recebidos: {latest_data}")

@app.route('/data')
def get_data():
return jsonify(latest_data)

def run_flask():
app.run(host='0.0.0.0', port=5000)

if __name__ == '__main__':
# Iniciar o servidor Flask em uma thread separada
flask_thread = threading.Thread(target=run_flask)
flask_thread.daemon = True
flask_thread.start()

# Conectar à balança
# (código de conexão omitido por brevidade)

try:
while True:
time.sleep(1)
except KeyboardInterrupt:
print("Encerrando...")

Com este servidor rodando, você pode acessar http://localhost:5000/data para obter os dados mais recentes da balança em formato JSON.

Nota: A Xiaomi não fornece uma API pública para acessar os dados da balança diretamente.

As soluções apresentadas aqui são baseadas engenharia reversa e podem não funcionar em todos os casos.
