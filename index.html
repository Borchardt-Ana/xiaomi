Problemas para Conectar com a Xiaomi Mi Scale 2?

Soluções e alternativas para acessar os dados da sua balança

Por que não consigo encontrar minha balança?

A Xiaomi Mi Scale 2 usa um protocolo Bluetooth proprietário que nem sempre é compatível com a Web Bluetooth API. Aqui estão as principais razões:

  • Problemas de compatibilidade: A Web Bluetooth API ainda não é totalmente suportada por todos os navegadores
  • Protocolo proprietário: A Xiaomi usa um protocolo customizado que requer autenticação específica
  • Requisitos de segurança: A conexão Bluetooth requer HTTPS (exceto em localhost)
  • Permissões: Seu navegador pode estar bloqueando o acesso ao Bluetooth
  • Balança não descoberta: A balança pode não estar em modo de descoberta

Soluções Alternativas

1. Usar Aplicativo Intermediário

Use aplicativos como "Tools & Mi Band" para Android que podem exportar dados para Google Fit.

Em seguida, use a API do Google Fit para acessar os dados.

Baixar App

2. Acesso Root no Android

Com acesso root, você pode extrair o banco de dados do aplicativo Mi Fit.

Os dados são armazenados em um arquivo SQLite que pode ser exportado.

3. Script Python com Bluetooth

Use um script Python no computador para se conectar via Bluetooth e extrair os dados.

Requer conhecimentos de programação e Bluetooth LE.

Configuração para Web Bluetooth

Se ainda quiser tentar a Web Bluetooth, siga estas etapas:

  1. Use o Chrome para Android ou Edge Chromium
  2. Certifique-se de que a página está sendo servida via HTTPS (ou localhost)
  3. Ative o Bluetooth do seu telefone
  4. Abra as configurações do Chrome → Configurações do site → Bluetooth → Permitir
  5. Mantenha a balança próxima e ligada
  6. Fique na balança para ativar o modo de descoberta

Nota: A Xiaomi não fornece uma API pública para acessar os dados da balança diretamente.

As soluções apresentadas aqui são alternativas não-oficiais.
